\doxysubsection{Detailed Description}
busqueda binaria 

Las opciones son las siguientes:\+
\begin{DoxyItemize}
\item 1) Crear un vector con tamaño de n elementos.
\item 2) Buscar un número a partir de la versión iterativa del algoritmo de búsqueda binaria.
\item 3) Buscar un número a partir de la versión recursiva del algoritmo de búsqueda binaria.
\item 4) Salir del programa. 
\end{DoxyItemize}
